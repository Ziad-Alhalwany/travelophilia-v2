# CRM → Ops Handoff (بعد Booked)

## 1) متى يتم الـHandoff؟
- بعد تأكيد الدفع/العربون + موافقة العميل على Scope النهائي.

---

## 2) ما الذي يُسلّم للأوبريشنز؟
- بيانات العميل الأساسية + وسيلة تواصل
- تواريخ الرحلة + العدد
- تفاصيل الإقامة/النقل/الأنشطة المتفق عليها
- أي طلبات خاصة (حساسية/خصوصية/ملاحظات)
- سياسة الإلغاء التي تم إبلاغ العميل بها (Yes/No)
- Voucher draft أو تفاصيل الحجز للـVendor
