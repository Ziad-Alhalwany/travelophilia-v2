# Finance Overview — Travelophilia

## الهدف
تسعير “مربح + قابل للبيع” مع حماية من التسعير الغلط اللي يهدّ المشروع.

## المبادئ
- **الربح لازم يبقى Guardrailed**: مفيش بيع تحت Floor.
- **العميل يشتري طمأنينة**: السعر لازم يشرح “القيمة” مش رقم فقط.
- **أي خصم = قرار** (مش رد فعل): لازم سبب + حد أقصى + موافقة.

## مخرجات إلزامية (Finance Deliverables)
- Cost model (Direct + Variable + Overhead allocation)
- Pricing floors + margin guardrails
- Packages (Starter/Plus/Premium)
- Discount policy
- Refund/Cancellation policy
- KPIs: AOV / Gross margin / Contribution margin / Refund rate
