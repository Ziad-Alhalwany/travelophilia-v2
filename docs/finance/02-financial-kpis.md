# 02-financial kpis + Unit Economics

## 1) التعريفات
- **AOV**: متوسط قيمة الطلب
- **GM (Gross Margin)**: (Revenue - Direct Costs) / Revenue
- **CM (Contribution Margin)**: (Revenue - Direct Costs - Variable Ops/Payment Fees) / Revenue

## 2) Guardrails (حدود أمان مقترحة)
- GM الحد الأدنى: 20% (Trips عادية)
- GM للـCustom: 25%+ (مخاطر أعلى)
- Fees (بوابة دفع/تحويل): تُحسب ضمن CM
- Refund risk: تُحسب ضمن Buffer

## 3) قرار السعر
- السعر النهائي لازم يحقق:
  - Floor price (عدم البيع تحت التكلفة)
  - Margin target
  - قابلية بيع (marketing angle)
