# Personas — شرائح العملاء

## 1) المسافر السريع
- عايز اختيار جاهز (Templates)
- حساس للوقت
- يحتاج: خيارات قليلة + خطوات واضحة

## 2) محب التفاصيل
- عايز تخصيص
- يسأل كتير
- يحتاج: شفافية + Option A/B

## 3) قائد مجموعة (Group Leader)
- يهتم بالتنظيم
- يحتاج: مرونة + قواعد واضحة + تأكيد كتابي

## 4) Private/Luxury
- حساس للخصوصية
- يحتاج: ثقة + خدمة أسرع + تفاصيل دقيقة
