# Funnel Map — خريطة التحويل (MVP)

## الهدف

تحويل الزيارة إلى Lead ثم Booking بأقل احتكاك.

---

## 1) مراحل الفانل

### A) اكتشاف (Discover)

- صفحات: Home / Destinations / Trips
- هدف: اختيار سريع + ثقة أولية

### B) إقناع (Consider)

- Trip Details
- هدف: تقليل شك العميل + وضوح “الخطوة الجاية”

### C) طلب (Request)

- Custom Trip Request / Reserve
- هدف: فورم قصير + تأكيد فوري

### D) تشغيل (OpsCRM)

- تواصل واتساب + Option A/B
- هدف: إرسال عرض خلال SLA

### E) دفع/تأكيد

- عربون/دفع
- هدف: تأكيد واضح + Voucher

---

## 2) CTA في كل مرحلة

- Discover: “شوف الرحلات”
- Consider: “اطلب رحلتك”
- Request: “ابعت الطلب”
- Ops: “اختار A/B”
- Payment: “أكد الحجز”
