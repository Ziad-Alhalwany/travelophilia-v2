# Offers & Packages — صياغة العروض (MVP)

## 1) مبدأ العرض
- مش سعر وبس: “قيمة + راحة + متابعة + وضوح”
- دايمًا نعرض 2–3 اختيارات: Starter / Plus / Premium أو Option A/B

## 2) Offers جاهزة
- “هنطلع لك خيارين واضحين بدل الزحمة”
- “متابعة خطوة بخطوة لحد التأكيد”
- “كل حاجة مكتوبة: يشمل/لا يشمل”

## 3) Upsell بدون إزعاج
- Add-ons تظهر كاختيارات: “تحب تضيف؟”
- هدفها رفع AOV مش تعقيد القرار
