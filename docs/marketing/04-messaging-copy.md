# Messaging & Copy — نبرة الكلام (Tone)

## 1) النبرة الأساسية
- إنساني + واثق + عملي
- بدون مبالغة
- وعد واضح: “كل خطوة واضحة”

## 2) جُمل جاهزة
- “تمام… خلّيني أرتب لك خيارين مناسبين”
- “ده يشمل كذا… وده لا يشمل كذا”
- “قبل ما تأكد، كل حاجة هتبقى مكتوبة”

## 3) ممنوعات
- وعود كبيرة بدون دليل
- كلام كتير بدون CTA
- مصطلحات صعبة
