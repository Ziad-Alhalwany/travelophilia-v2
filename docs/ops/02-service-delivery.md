# Service Delivery SOP — من الدفع إلى نهاية الرحلة

## 1) بعد الدفع مباشرة
- تأكيد Scope النهائي (Hotel/Transport/Activities)
- إرسال “تأكيد استلام + الخطوة القادمة”
- فتح Booking ID

---

## 2) تأكيد الموردين (Vendor Confirmation)
- فندق/شاليه: availability + الأسعار + سياسة الإلغاء
- نقل: سائق/شركة + مواعيد + pickup point
- أنشطة: وقت + مكان + شروط

---

## 3) تجهيز Voucher + Itinerary
- Voucher واضح: الاسم + التواريخ + الخدمات + أرقام الطوارئ
- Itinerary بسيط (حتى لو نسخة أولى): يوم/يوم

---

## 4) Pre-trip Briefing (قبل الرحلة)
- نقطة التجمع + وقت التحرك
- checklist: بطاقة/هوية/ملابس/حاجات مهمة
- “إزاي تتواصل معانا” أثناء الرحلة

---

## 5) أثناء الرحلة
- متابعة ناعمة (مرة يوميًا) “هل كل شيء تمام؟”
- التعامل مع المشاكل عبر incident playbook

---

## 6) بعد الرحلة
- رسالة شكر + Feedback
- اقتراح رحلة لاحقة أو Referral
