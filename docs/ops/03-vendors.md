# Vendors Playbook — اختيار وتقييم الموردين

## 1) معايير اختيار الفندق/الشاليه
- تقييمات قوية + صور حقيقية + موقع مناسب
- سياسة إلغاء واضحة
- مرونة في التعديل (لو أمكن)
- سرعة رد

---

## 2) معايير اختيار النقل
- التزام بالمواعيد
- سيارة نظيفة + أمان
- سائق محترم
- بديل جاهز لو حصل عطل

---

## 3) تسجيل Vendor Profile (حد أدنى)
- الاسم + وسيلة التواصل
- الأسعار الأساسية
- سياسات الإلغاء
- نقاط قوة/ضعف
- ملاحظات تشغيل

---

## 4) Vendor Score (شهريًا)
- On-time
- جودة الخدمة
- شكاوى
- مرونة
- السعر مقابل القيمة
