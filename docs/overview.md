# Overview — Travelophilia

## 1) Vision

Travelophilia مش “موقع رحلات” وبس… ده Experience + Operations System.

## 2) Golden Journey

اختيار → مقارنة → تخصيص → طلب → متابعة CRM → دفع/تأكيد → برنامج رحلة

## 3) Tech Stack

- Frontend: React (Vite)
- Backend: Django (DRF)
- DB: PostgreSQL

## 4) Repo Structure (مختصر)

- `src/` → Frontend
- `backend/` → Django API
- `docs/` → Product/Marketing/Analytics/Ops/CRM/Finance/Pricing
- `.agent/` → rules/workflows/templates

## 5) Conventions

- Backend: snake_case
- Frontend: camelCase
- Bridge: converter عام + mappers قليلة
