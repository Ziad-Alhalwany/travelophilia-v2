# UI Copy Guidelines — Travelophilia

## 1) Tone

- إنساني + هادي + واثق
- بسيط ومباشر
- “ليه بنسأل؟” دايمًا

## 2) ممنوعات

- وعود مبالغ فيها
- مصطلحات صعبة
- ضغط زائد

## 3) قواعد UX Copy

- كل CTA يوضح الخطوة القادمة
- رسائل الخطأ: تقول “إيه المشكلة + إزاي تصلحها”
- رسائل النجاح: تقول “اتسجل + إمتى نرد + إيه التالي”

## 4) أمثلة جاهزة

### CTA

- "اطلب رحلتك الآن"
- "خلّينا نجهز لك خيارين واضحين"

### Success

"تم استلام طلبك ✅ هنرد عليك خلال {X} دقيقة برسالة واتساب."

### Error

"من فضلك اكتب رقم واتساب صحيح (مثال: 01xxxxxxxxx)."
