# User Journey — Travelophilia

## 1) Persona مختصر

- مسافر سريع: عايز اختيار جاهز
- محب للتفاصيل: عايز تخصيص
- Group leader: تنظيم جماعي
- Luxury/Private: خصوصية وجودة

---

## 2) الرحلة داخل الموقع (Golden Path)

### Step 1: دخول

مصدر: إعلان/سوشيال/SEO  
هدفنا: وضوح + ثقة + CTA واضح

### Step 2: Browse Trips/Destinations

هدف المستخدم: “أشوف خيارات”  
هدفنا: “أختصر له القرار”

### Step 3: Trip Details

هدف المستخدم: “هل أثق؟ هل ده يناسبني؟”  
هدفنا: “تفاصيل + ضمانات + خطوات”

### Step 4: Request Booking / Custom Trip

هدف المستخدم: “أبعت طلب بدون تعقيد”  
هدفنا: “Form بسيط + طمأنة + وقت رد متوقع”

### Step 5: CRM Follow-up

هدف المستخدم: “رد سريع”  
هدفنا: “تأهيل بسيط + عرضين واضحين”

### Step 6: Payment / Confirmation

هدف المستخدم: “أمان”  
هدفنا: “سياسة واضحة + تأكيد كتابي”

### Step 7: Itinerary + Voucher

هدف المستخدم: “أحس إنها اتقفلت”  
هدفنا: “ملف واضح + دعم وقت الرحلة”
